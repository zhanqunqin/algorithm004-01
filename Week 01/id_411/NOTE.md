#####循环双端队列
1. 数组实现的队列  

要点：  
    - 数组下标是离散型数字，呈现出有序递增特性，需要用模运算，使下标取值限制在[0, length -1]区间，为了形象表达，才采用环形图示，其本质是线性折叠。  
    - 对数组的操作  
      - 增  
        addToHead() 头部位置入队
        addToTail() 尾部位置入队
      - 删
        delFromHead() 头部位置出队
        delFromTail() 尾部位置出队
      - 改
        clear() 清空整个队列
      - 查
        length() 查询有效长度
        getHead() 获取头部位置元素
        getTail() 获取尾部位置元素
        isFull() 判断是否满列
        isEmpty() 判断是否空列

针对判断满列与空列的实现方式：
1. 采用 size = [0, length - 1] 标记
2. 采用 tag 标记出队/入队操作，frontIndex === tailIndex 共同判断
3. 数组预留多一位空间作为标记

个人感觉3的性能会比1/2高，因为数组下标访问的速度比较快些。
